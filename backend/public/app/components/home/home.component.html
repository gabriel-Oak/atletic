O que é o Atletic?

O Atletic é um sistema de gerenciamento de atletas olimpicos. Surgiu de uma grande iniciativa bilionária. Investidores anonimos deram então origem ao projeto Atletic. O objetivo da plataforma formada nos meados de 2019, é, como o próprio nome ja diz: Gerenciar de forma atlética, competidores olímpicos. Sua filosofia pregra a transparência e praticidade para com os usuários.

Por Quê Atletic?

Com o Atletic, você só tem a ganhar. Economise tempo, juntando todos os seus atlétas em um lugar só. Praticiade e facilidade de gerenciamento da plataforma, entregam a você a melhor experência possível como usuário. Mas é o bolso? Como fica? Você não paga nada para utilisar o Atlétic, a plataforma é completamente aberta!

Como Usar Atletic?

Atletic foi pensado para ser de facil utilisação. Basta selecionar a aba "Modalidade" caso a área de atuação do seu competidor ainda não exista em nossos registros. Após, clique no botão Novo para inserir um registro. Caso deseje alterar uma Modalidade clique sobre o nome dela, e altere os registros. O botão irá apagar o registro, então cuidado.

Para atletas o procedimento é o mesmo, com a única diferença de que você devera selecionar a modalidade para ele dentre as cadastradas.
